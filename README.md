### ✨ Sobre o projeto
Este é o meu portfólio digital completo, desenvolvido para apresentar meus trabalhos, habilidades e trajetória na área de tecnologia. Criei uma interface moderna, atraente e totalmente funcional, com foco total em **experiência do usuário, design responsivo e alta performance**.

O projeto segue os princípios de UI/UX Design, com navegação intuitiva, carregamento rápido e adaptação perfeita para qualquer tela — celular, tablet ou computador.

**Conceitos e boas práticas:**
